SUCHAI
======

Software de vuelo del Cubesat SUCHAI
_SUCHAI Cubesat Flight Software_

Key Features:
* Designed for medium-range microcontrollers, as Microchip PIC24.
* Low memory foot print
* Highly extensible command processor architecture
* Powered by FreeRTOS
